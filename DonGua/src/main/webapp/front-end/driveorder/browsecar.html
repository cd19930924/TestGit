是誰需要「無障礙環境」？

我們以「無障礙環境的全面連結」做為服務宗旨， 讓行動不便的朋友也能真正掌握與安排自己「行的需求」。

預約接駁

服務方式

  1. 服務對象為行動不便者，外出需要輪椅直接上下車的使用者。
  2. 攙扶行動不便者上下輪椅， 協助輪椅使用者上下車，並接送至目的地。
  3. 輪椅可直接上下車 。
  4. 每趟車資為 : 里程費用+服務費85元 。
